# Memact

Memact is a personal address (e.g., username.memact.com) where approved things about you accumulate over time.

Instead of maintaining a manual profile or managing a dashboard, your identity grows naturally from your usage of apps, agents, and services.

## Core Architecture
- **Website**: Dynamic narrative and simple dashboard for reviewing suggestions.
- **Access**: Gateway that handles app requests, filters duplicate/low-info spam, and checks permissions.
- **Wiki (Notebook)**: Rules and visibility settings (Public, Friends, Only Me).
- **Context**: Standard shapes and categories for suggested statements.
- **Memory**: Secure storage and search index.
- **SDK**: Package for apps to propose updates and query allowed details.
